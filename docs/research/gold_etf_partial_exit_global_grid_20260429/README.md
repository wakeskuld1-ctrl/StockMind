# 黄金 ETF D15分段减仓 + 高点回撤穷举优化

## 研究契约

- 标的：`518800.SH`
- 数据：沿用冻结黄金母体信号与 ETF 历史
- 资金基准：`1,000,000`
- 入场规则：不变
  - 第一层：母体信号后 `T+1` 开盘买入 `50%`
  - 第二层：相对第一层锚价再跌 `-5%`，且母体再次触发，`T+1` 开盘加 `40%`
  - 不做第三层
- 早期失败保护：保留 `5D` 未修复则下一交易日开盘退出
- 本轮只优化退出：
  - `D15` 是否先减仓
  - 剩余仓位用高点回撤退出
  - `D15` 仍亏损时如何处理
- 结论边界：只代表本参数空间内的穷举最优，不代表市场真实全局最优。

## 参数空间

- `D15减仓比例`：`0% / 30% / 50% / 70%`
- `D15减仓条件`：`always / D15收益>0 / D15收益>1% / D15收益>2%`
- `剩余仓位高点回撤`：`1.0% / 1.2% / 1.5% / 1.8% / 2.0% / 2.5% / 3.0%`
- `高点回撤启动日`：`D12 / D15 / D18`
- `最大持有日`：`20D / 30D / 45D / 60D`
- `D15亏损处理`：
  - `time_exit_20d`
  - `no_partial_trailing`
  - `hold_to_max`

总有效组合：`3276`

## 最优结果

最优配置：

`p0.70_canchor_return_gt_0.01_dd0.010_s18_h60_loss_hold_to_max`

含义：

- 如果 `D15` 收益 `> 1%`：下一交易日开盘卖出 `70%` 的当前策略仓位
- 剩余 `30%`：从 `D18` 开始跟踪高点
- 从高点回撤 `1.0%`：下一交易日开盘清仓
- 如果 `D15` 仍未达到 `1%` 收益：不做 D15 减仓，最多持有到 `60D`

## 关键解释

这次算法没有证明“D15固定卖50%”是最优。

它给出的更强结论是：

> `D15` 已经有明确修复利润时，先大比例落袋；剩余小仓位用紧高点回撤继续吃尾巴。  
> `D15` 仍不理想时，不急着割掉，而是给到更长修复窗口。

也就是说，收益提升主要来自两部分：

1. 盈利样本：`D15 > 1%` 后先卖 `70%`，降低利润回吐。
2. 未充分修复样本：不机械 `20D` 退出，而允许最多观察到 `60D`。

## 稳定性观察

Top25 里比较集中的特征：

- `D15收益 > 1%` 是最常见的减仓条件。
- `70%` 减仓比例出现最多，但 `50% / 30%` 也进入前列。
- `hold_to_max` 是 Top25 的共同特征，说明 `D15亏损时延长修复窗口` 是本次优化的重要收益来源。
- 最大持有 `60D` 在 Top25 中占主导。

## 风险

- 本轮是历史参数空间穷举，存在过拟合风险。
- `hold_to_max=60D` 会拉长资金占用时间，虽然历史回撤没有扩大，但实盘心理压力会更大。
- 最优规则比当前正式规则复杂，需要每日跟踪：
  - D15收益
  - 是否已部分卖出
  - 剩余仓位高点
  - 从高点回撤幅度
  - 最大持有日

## 下一步建议

不要直接上线最优参数，先做稳定性复核：

1. 只围绕 Top 区域做窄域复核：
   - 减仓比例：`50% / 60% / 70%`
   - D15条件：`>0.5% / >1% / >1.5%`
   - 高点回撤：`0.8% / 1.0% / 1.2% / 1.5%`
   - 最大持有：`45D / 60D`
2. 做年份切片，确认不是由少数年份贡献。
3. 如果切片稳定，再升级成“候选正式规则”。
