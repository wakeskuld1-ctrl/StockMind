# 黄金 ETF 15D收益锚点延续退出研究

## 研究契约

- 标的：`518800.SH`
- 资金基准：`1,000,000`
- 入场规则：沿用已验证两层规则，不改入场、不改仓位
- 第一层：黄金母体信号触发后，`T+1` 开盘买入 `50%`
- 第二层：相对第一层锚价再跌约 `-5%`，且黄金母体再次触发，`T+1` 开盘加到 `40%`
- 第三层：不允许
- 早期失败保护：保留 `5D` 未修复则下一交易日开盘退出
- 新研究点：从 `15D` 开始，把 `15D` 收益作为锚点；后续若收盘收益跌破该锚点，下一交易日开盘退出
- 最大持有扫描：`20D / 30D / 40D / 60D / 90D / 120D / 160D / 200D`

## 结论

`15D收益锚点延续` 不建议升级为正式规则。

原因不是 200D 不够长，而是这个锚点退出条件太紧：大部分交易在 `D16/D17` 就因为跌破 `15D` 收益锚点退出，真正延续到 `40D` 以上的样本很少。拉长到 `200D` 后，收益没有提高，收益回撤比反而低于 `20D`。

当前更稳的正式规则仍应保持：

- 第二层：`-5%` + 母体再次触发 + `40%`
- 不做第三层
- 保留 `5D` 失败保护
- 主修复窗口继续按 `10D~15D` 观察
- 固定 `20D` 退出仍优于本次测试的 `15D锚点延续`
