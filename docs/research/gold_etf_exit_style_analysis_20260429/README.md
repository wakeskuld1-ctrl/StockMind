# 黄金 ETF 15D以后退出方式数据分析

## 研究契约

- 标的：`518800.SH`
- 数据区间：沿用冻结 ETF 历史与黄金母体信号
- 资金基准：`1,000,000`
- 入场规则：不变
  - 第一层：母体信号后 `T+1` 开盘买入 `50%`
  - 第二层：相对第一层锚价再跌 `-5%`，且母体再次触发，`T+1` 开盘加 `40%`
  - 不做第三层
- 早期失败保护：保留 `5D` 未修复则下一交易日开盘退出
- 本轮只研究 `15D以后` 的退出方式
- 最大观察上限：`60D`

## 三类退出规则

1. 高点回撤退出
   - 从 `D15` 后继续跟踪持仓内最高收盘收益
   - 从高点回撤 `1% / 1.5% / 2% / 3%` 时，下一交易日开盘退出

2. MA趋势退出
   - 从 `D15` 后继续持有
   - 收盘跌破 `MA5 / MA10 / MA20` 时，下一交易日开盘退出

3. 分段止盈退出
   - 从 `D15` 后观察是否进入 `3% / 5% / 8%` 收益区
   - 进入收益区后，若利润回吐 `30% / 50%`，下一交易日开盘退出

## 分族结果

| 类型 | 最佳规则 | 累计收益 | 最大回撤 | 收益回撤比 |
|---|---|---:|---:|---:|
| 高点回撤 | `hwm_dd_0.015` | 109.80% | -14.27% | 7.69 |
| 分段止盈 | `profit_0.050_giveback_0.50` | 98.42% | -13.19% | 7.46 |
| MA趋势 | `ma5_trend` | 84.84% | -14.27% | 5.94 |

## 关键观察

### 1. 高点回撤是最值得细化的方向

`hwm_dd_0.015` 是本轮第一名：

- 累计收益：`109.80%`
- 最大回撤：`-14.27%`
- 收益回撤比：`7.69`
- 平均持有：`23.43` 个交易日

这说明 `15D以后` 并不是完全不能吃，但不能靠固定延长持有，而应该靠“从阶段高点回撤后退出”来保护利润。

### 2. 分段止盈降低回撤，但收益不够强

`profit_0.050_giveback_0.50` 的最大回撤最低：

- 累计收益：`98.42%`
- 最大回撤：`-13.19%`
- 收益回撤比：`7.46`

它适合做风险控制候选，但收益低于高点回撤，也低于当前两层固定 `20D` 最优规则。

### 3. MA趋势暂时不值得作为主线

MA 类最优 `ma5_trend` 只有：

- 累计收益：`84.84%`
- 最大回撤：`-14.27%`
- 收益回撤比：`5.94`

MA退出在这套黄金 ETF 均值修复规则里偏钝，不适合作为下一轮主优化方向。

## 与当前正式规则的关系

当前两层固定 `20D` 最优规则历史结果约为：

- 累计收益：`112.75%`
- 最大回撤：`-14.27%`
- 收益回撤比：`7.90`

本轮最佳 `hwm_dd_0.015`：

- 累计收益：`109.80%`
- 最大回撤：`-14.27%`
- 收益回撤比：`7.69`

因此，本轮数据分析的结论是：

> 高点回撤退出有价值，值得细化；但当前版本还不能替换正式规则。

## 下一步细化建议

只细化高点回撤，不继续扩 MA。

建议下一轮围绕 `hwm_dd_0.015` 做小范围扫描：

- 回撤阈值：`1.2% / 1.5% / 1.8% / 2.0%`
- 启动日：`D12 / D15 / D18`
- 最大持有：`30D / 45D / 60D`
- 可选保护：若 `D15` 仍为负收益，不进入高点回撤延续，直接按原规则退出

验收条件：

- 必须超过当前固定 `20D` 规则的收益回撤比 `7.90`
- 或者在收益不低于 `105%` 的前提下，把最大回撤压到 `13%` 左右
- 否则不升级正式规则
