# 黄金 ETF D15未达标分支与资金效率复核

## 问题

上一版候选规则存在一个执行疑问：

> 如果 `D15` 没有达到 `+1%`，但 `D16/D17/D20` 后来达到，是否应该卖出？  
> 如果一直没达到，是否真的要持有到 `D60`？这样 IRR 会不会很差？

本轮专门测这个分支。

## 测试规则空间

- `D15` 后达标窗口：只看 `D15` / 滚动到 `D18` / 滚动到 `D20`
- 达标阈值：`0.5% / 1.0% / 1.5%`
- 达标后减仓比例：`50% / 60% / 70%`
- 剩余仓位高点回撤：`0.8% / 1.0% / 1.2% / 1.5%`
- 剩余仓位高点回撤启动：达标日当天 / 达标后 2 个交易日
- 未达标退出日：`D20 / D30 / D45 / D60`

有效组合：`864`

## 最优结果

最优配置：

`p0.70_thr0.010_win15_dd0.012_off2_unmet60`

含义：

- 只看 `D15` 是否达到 `+1%`
- 如果达到：下一交易日卖出 `70%`
- 剩余 `30%`：从达标后 2 个交易日开始跟踪高点
- 从高点回撤 `1.2%`：下一交易日清仓
- 如果 `D15` 未达到 `+1%`：继续持有到 `D60`

## 关键结论

### 1. D15以后滚动达标没有胜出

Top25 里：

- `anchor_window_end_day = 15`：25 个
- `anchor_window_end_day = 18`：0 个
- `anchor_window_end_day = 20`：0 个

这说明历史数据里，最优不是“D15没达标后继续等到D20，只要达标就卖”，而是：

> D15 达标就锁利；D15 没达标就不要急着卖，继续给修复时间。

### 2. D15未达标后，D60仍然胜出

Top25 里：

- `unmet_exit_day = 60`：25 个
- `D20/D30/D45`：0 个

这说明按当前数据，D15未达标后提前退出会损失后续修复收益。用户担心的 IRR 问题是合理的，但历史结果显示，延长到 D60 带来的收益补偿仍然更强。

### 3. 最优规则略有修正

上一版高点回撤是 `1.0%`。本轮加入资金效率后，最优变成：

- 剩余仓位高点回撤 `1.2%`
- 从达标后 `2` 个交易日开始跟踪

这比 `1.0%` 略宽一点，减少了被短期波动洗出的概率。
