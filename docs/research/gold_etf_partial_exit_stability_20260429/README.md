# 黄金 ETF 分段减仓最优线稳定性拆解

## 拆解目标

本轮只拆解上一轮最优线：

`D15收益>1% -> 卖出70% -> 剩余30%从D18开始按1%高点回撤清仓 -> D15未达标则最多持有60D`

目标不是继续扩大参数，而是确认：

- 是否只是单个参数点碰巧最好
- 年度收益是否集中在少数年份
- 这条线是否值得进入候选正式规则

## 窄域参数

围绕 Top 区域做小范围复核：

- `D15减仓比例`：`50% / 60% / 70%`
- `D15减仓条件`：`收益>0.5% / 收益>1% / 收益>1.5%`
- `剩余仓位高点回撤`：`0.8% / 1.0% / 1.2% / 1.5%`
- `高点回撤启动日`：`D15 / D18`
- `最大持有日`：`45D / 60D`
- `D15未达标处理`：`hold_to_max`

有效组合：`144`

## Top25 参数集中度

### 减仓比例

- `70%`：11 个
- `60%`：8 个
- `50%`：6 个

解释：`70%` 最强，但 `50% / 60%` 也有效，说明“D15盈利后先减大半”比某一个固定比例更重要。

### D15减仓条件

- `D15收益>1%`：17 个
- `D15收益>0.5%`：6 个
- `D15收益>1.5%`：2 个

解释：`D15收益>1%` 是最稳定的触发线。太宽容易提前卖，太严会错过锁利。

### 高点回撤阈值

- `0.8%`：3 个
- `1.0%`：5 个
- `1.2%`：6 个
- `1.5%`：11 个

解释：最优点是 `1.0%`，但 `1.2% / 1.5%` 也很强。实盘若担心被短波动洗出，可以考虑 `1.2%~1.5%` 作为稳健版。

观察：

- 最弱年份是 `2016`，仍为小幅正收益。
- `2026` 是未完整年份，只能作为阶段观察，不能等同完整年度结论。

## 结论

这条线值得进入候选正式规则。

原因：

1. 窄域不是单点有效，`92.36%` 组合超过当前正式规则门槛。
2. Top10 的收益回撤比集中在 `9.50~9.88`，明显高于当前参考门槛 `7.90`。
3. 年度拆解没有出现“只靠一年爆发”的情况。

但我仍不建议今天直接替换正式规则。下一步应把它整理成候选执行规则，并用当前持仓状态做一次实盘路径映射：

- 当前持仓进入第几个交易日
- 哪天是策略意义上的 `D15`
- `D15收益>1%` 如何按你的综合成本和策略锚价计算
- 如果触发，卖出多少份
- 剩余份额如何记录高点和回撤
