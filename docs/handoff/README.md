# Handoff Docs

This directory stores handoff and resume material.

Put documents here when they are meant to help the next AI or engineer continue work with low context cost.

Use this directory for:

- branch-local status snapshots
- handoff procedures
- unresolved handoff or standardization issues
- historical resume notes that still help the next session

Do not use this directory for stable governance source-of-truth files. Those belong under `docs/product/` and `docs/governance/`.
